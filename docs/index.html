FAQ

Made for uptime, status and server management

How does UptimeBot reduce alert spam?

Smart notification timing and auto-expiring menu sessions keep channels readable while still reporting important status changes.

What does UptimeBot actually do?

UptimeBot monitors the bots you add to your watchlist and posts clear online and offline updates in your configured alert channel. It helps staff quickly see status changes without manual checking.

How do I start tracking bots in my server?

Run /setpingchannel to choose where alerts go, then open /watchlist to add bots by name or ID. Once added, UptimeBot handles uptime and downtime notifications automatically.

How long is server data kept?

If UptimeBot is removed from a server, that server's configuration and watchlist data is automatically deleted after 7 days. If the bot is re-added before 7 days, the process will be canceled.
